# Sandhill Springs Investment Logo Kit

This folder contains logo assets prepared for web, social, and print use.

## Folder structure

- `vector/` - primary source logos (SVG, scalable for print and design tools)
- `raster/` - ready-to-use PNG exports
- `social/` - LinkedIn profile/banner SVG masters
- `print/` - name card template SVG files (with bleed/trim guides)

## Recommended logo files

- Primary horizontal logo: `vector/shsi-horizontal-color.svg`
- Reverse horizontal logo (for dark backgrounds): `vector/shsi-horizontal-white.svg`
- Icon / avatar mark: `vector/shsi-mark-color.svg`
- Stacked logo (square placements): `vector/shsi-stacked-color.svg`
- Wordmark only: `vector/shsi-wordmark-color.svg`

## Name card usage

- Editable front: `print/name-card-front.svg`
- Editable back: `print/name-card-back.svg`

Card template sizing:

- Bleed size: 3.75 x 2.25 inches
- Trim size: 3.5 x 2.0 inches
- Safe area: shown as dashed box in template

## Brand colors

- Deep navy: `#122531`
- Evergreen: `#1F4F4B`
- Sand accent: `#BF9863`
- Light sand accent: `#F9E1BA`
- Slate text: `#40535F`

## Notes

- Use SVG files for all print production and professional editing.
- Use PNG files for social uploads and quick sharing.
- The name card templates include placeholder contact details and can be edited directly in Figma, Illustrator, or Inkscape.
